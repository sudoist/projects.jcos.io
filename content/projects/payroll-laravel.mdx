## Summary

The requirement for this project was for the employees of a manpower agency to encode their employees then to be able to upload payslips.

Admin are also able to upload employee files to 201 file.

The project has been used for a couple of years until recently.

- Web app built with [Laravel](https://laravel.com/)
- CSS framework with [Tailwind](https://tailwindcss.com)
- Deployed on [DigitalOcean](https://www.digitalocean.com/)
