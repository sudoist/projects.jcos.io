## Summary

A little easier project to have a nicer display of the data from a Python/Flask API.

The web application accepts filters then sends it to the API to fetch data for display.

One of the most challenging parts of this project is to display the data in terms of Regions and Provinces for the Philippines map.

This was accomplished using Chart.js Chart Geo.

Please see pens below:

[REGIONS](https://codepen.io/sudoist/pen/wvGmVRZ)
